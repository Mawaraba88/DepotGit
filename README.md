# Dépôt Local
Un dépot local est un dossier que Git versionnalise.
Ce dossier contiendra les différents commits

## Initialisé un dépot
- On creer un dossier avec la commande mkdir
- Avec la commande git init on initialise un dépot

![](dépot_local.png)
